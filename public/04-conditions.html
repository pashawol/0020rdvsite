Условия подписки:

  • Простая смена тарифа – просто сообщите менеджеру, какой тариф хотите использовать сейчас
  • Все тарифы включают в себя дополнительного пользователя для технического подключения и интеграции системы

Подписка включает в себя:

  • Демонстрацию возможностей сервиса
  • Регулярные обновления для всех пользователей
  • Поддержку изменений API со стороны маркетплейсов
  • Оперативную техническую поддержку и консультации по работе сервиса
